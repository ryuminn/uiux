마지막 우선순위

1. div는 의미 없는 빈그릇 태그로 뭔가를 감싸는 컨테이너 기능 / division의 약자 > CSS화면 디자인을 위해 인위적으로 만들어진 태그
2. 제목태그 h1은 페이지에 딱 한번만 쓰기를 권장 단, h2-h6은 여러번 중복가능
3. br태그는 문장과 문장의 한줄 바꿈 기능
4. br태그를 여러줄 떨어뜨리기 위해서 여러번 중복 사용은 비권장
5. hr태그는 내용과 내용을 선으로 구분지을때 사용
6. p태그는 단락을 생성
7. 빈그릇 태그 span은 특정 텍스트에 css효과를 적용할때 사용
