ui/ux 디자인의 개념

UI의 정의

  • 사람과 시스템간의 접점
  • 사용자와 시스템 사이의 정보채널
  • 사용자에게 컴퓨터를 편리하게 사용할 수 있는 환경을 제공하는 설계 내용

User Interface의 기본원칙

  1. 유용성의 원칙 : 쓸모가 있어야 한다.
  2. 사용성의 원칙 : 사용하기 쉽고 편리해야 한다.
  3. 감성의 원칙 : 사용자의 마음을 즐겁고 기쁘게해야 한다.

UX(User eXperience) 사용자 경험(Bad/Good/Excellent)

만족도는 이성적/감성적인 피드백의 총체적인 이미지이며 아래의 3가지 요소에 의해서 좌우된다.

  1. Like : 얼마나 좋아하느냐 ?
  2. Understand : 얼마나 시스템을 이해하고 있느냐 ?
  3. Use : 얼마나 제품을 잘 이용할 수 있는가 ?
